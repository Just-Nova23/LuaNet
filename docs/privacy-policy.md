# LuaNet privacy policy

Last updated: 1 July 2026

LuaNet is an Android app for hosting Luanti servers on your own phone. This policy covers
the LuaNet app, the LuaNet website, and the NovaX public access control plane.

Contact: `luanet@novaxhosting.com`

## Summary

- LAN hosting works without an account and without advertising.
- LuaNet does not upload worlds, game chat, player names, installed mods, or console logs to
  NovaX.
- Public access and Premium require a Firebase-backed NovaX account.
- NovaX stores only the account, device, tunnel, entitlement, aggregate usage, and
  anti-abuse data needed to operate public tunnels and Premium limits.
- Public game packets pass through the NovaX relay in transit but are not retained.

## Data processed on the phone

LuaNet stores server profiles, worlds, games, mods, backups, settings, logs, player metadata,
and ContentDB downloads in app-private storage on the Android device. This local hosting data
stays on the phone unless the user exports it or shares it through Android.

The app may access network state, Wi-Fi state, foreground service, wake lock, notification,
and storage/document picker capabilities only to run local servers, keep them reachable, show
server status, import/export content, and notify the user.

## Data sent to NovaX

For public tunnels and Premium, the app sends NovaX:

- Firebase user identifier and verified sign-in provider state;
- device or installation identifier generated by the app;
- server profile identifier, tunnel hold/lease identifiers, assigned public port, lease
  state, lease timestamps, and hashed tunnel session credentials;
- Google Play purchase token for server-side Premium verification;
- aggregate operational usage needed for limits, abuse prevention, and capacity planning.

NovaX does not receive or store world files, chat, console logs, in-game player names, game
content packages, or local server configuration.

## Third-party services

LuaNet uses these services when the matching feature is used:

- Firebase Authentication for Google, GitHub, and email/password sign-in.
- Google Play Billing for Premium purchases and entitlement verification.
- Google AdMob and the Google User Messaging Platform for the Free public-start
  interstitial and EEA/UK consent flow.
- ContentDB for browsing and downloading Luanti games, mods, and texture packs.
- GitHub only when the user chooses GitHub sign-in or opens project links.
- Cloudflare/NovaX infrastructure for HTTPS delivery of the API and website.

Each provider may process data under its own terms and privacy notices.

## Advertising and analytics

LuaNet shows no advertisements for LAN hosting. Free public tunnel starts may show one
AdMob interstitial before the tunnel starts. Premium users skip public-start ads.

LuaNet does not run behavioral analytics. Crash reporting is disabled unless the user
explicitly opts in.

## Retention

- Account, entitlement, and active tunnel records are kept while the account exists or while
  needed to provide the feature.
- Stopped tunnel ports are kept only for the short grace period needed to prevent accidental
  reassignment during reconnects.
- Security and anti-abuse logs are retained for up to 30 days.
- Google Play/Firebase/AdMob/provider records follow the relevant provider retention rules.
- Local worlds and app data remain on the phone until the user deletes profiles, clears app
  data, or uninstalls LuaNet.

## Account and data deletion

Users can delete their NovaX account from the app. Deletion revokes active tunnels, removes
NovaX tunnel allocations and Premium entitlement records, and deletes the Firebase Auth user.
Security records may remain for up to 30 days where needed for abuse prevention or legal
obligations.

Users can also request deletion from the web by emailing `luanet@novaxhosting.com` with the
subject `LuaNet account deletion request`.

Deleting the NovaX account does not delete local worlds or backups on the phone. Those remain
under the user's local control.

## Children

LuaNet is intended for a Teen/13+ audience and is not directed to children. Users under the
minimum age required by their jurisdiction should not create a NovaX account or use public
tunnel features without appropriate permission.

## User rights

Depending on location, users may have rights to access, correct, delete, export, restrict,
or object to processing of their personal data. Send requests to `luanet@novaxhosting.com`.

## Security

LuaNet uses HTTPS/TLS for API traffic and keeps tunnel credentials temporary. The app never
asks users to disable Luanti mod security. Public tunnels carry traffic only for the selected
Luanti server port.

## Changes

This policy may be updated as LuaNet changes. The latest version is published on the LuaNet
website and reflected in the Play Console Data safety declaration.

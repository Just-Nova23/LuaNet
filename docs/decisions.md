# Locked product decisions

- Public package ID: `net.novax.luanet`.
- English-only beta; Teen audience and not directed to children.
- All ContentDB flags remain visible with warning badges.
- Incompatible packages may be installed only after explicit confirmation.
- New servers are unlisted and open; allowlist is optional.
- Free: one active server and a four-hour public maximum.
- LAN has no mandatory idle stop. Optional per-profile Auto off uses a configurable 1–1,440 minute no-player timeout and defaults to disabled.
- Premium: EUR 1.99 monthly or EUR 19.10 yearly, five active servers per account.
- Real purchases start with the Google Play beta; sideloaded builds use tester entitlements.
- Crash reporting is opt-in. Behavioral analytics are disabled.
- Restart after device reboot is always manual.
